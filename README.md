# 🚗 YOLOv5 License Plate Recognition

## 📌 Overview

This project implements an end-to-end real-time license plate recognition system using YOLOv5 and OCR.
It detects license plates from images or video streams and extracts the plate number automatically.

---

## 🛠️ Tech Stack

* Python
* YOLOv5
* OpenCV
* OCR (Tesseract / EasyOCR)

---

## 🎯 Features

* Real-time license plate detection
* Automatic text extraction (OCR)
* Video and image input support
* Bounding box visualization

## 🚀 How to Run

### 1. Clone repository

```bash
git clone https://github.com/your-username/yolov5-license-plate-recognition.git
cd yolov5-license-plate-recognition
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run detection

```bash
python detect.py --source path/to/video.mp4 --weights best.pt
```
